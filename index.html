🚀 Omnara Installation Guide

Transform Your AI Coding Assistant into a Collaborative Partner

What is Omnara?

Omnara is like a "remote control" for your AI coding assistants (Claude Code, Cursor, GitHub Copilot). Instead of sitting at your computer to interact with them, you can:

  • See what your AI is doing in real-time from your phone or web browser
  • Answer questions and guide your AI from anywhere
  • Get notifications when your AI needs help
  • Start coding sessions remotely

Think of it as turning your AI from a silent worker into a teammate you can talk to!

1Prerequisites (What You Need Before Starting)

Required Software:

  • Claude Code - The AI coding assistant from Anthropic
  • Python - A programming language (don't worry, you won't need to code!)
  • A Terminal/Command Line - Already on your computer
💡 Note for Mac Users: Python usually comes pre-installed. For Windows users, we'll show you how to install it.

2Installing Python (If Needed)

For Windows Users:

  1. Visit python.org/downloads
  2. Click the big yellow "Download Python" button
  3. Run the downloaded file
  4. IMPORTANT: Check the box that says "Add Python to PATH" before clicking Install

For Mac Users:

Open Terminal (find it in Applications > Utilities) and type:

python3 --version

If you see a version number (like "Python 3.9.6"), you're all set!

✅ Success Check: If Python is installed correctly, typing python --version or python3 --version in your terminal should show a version number.

3Installing Omnara (The Easy Part!)

Step-by-Step Installation:

1. Open Your Terminal/Command Prompt:

  • Windows: Press Windows Key + R, type "cmd", press Enter
  • Mac: Press Cmd + Space, type "Terminal", press Enter

2. Install Omnara with this simple command:

pip install omnara

Or if that doesn't work, try:

pip3 install omnara
💡 Tip: Copy and paste the command to avoid typos. Right-click to paste in Windows terminal, or Cmd+V on Mac.

3. Wait for Installation:

You'll see text scrolling by - this is normal! Wait until you see your cursor again (usually takes 30-60 seconds).

✅ Success: When complete, you should see something like "Successfully installed omnara-1.5.2"

4First-Time Setup (Authentication)

1. Start the authentication process:

omnara --auth

2. What happens next:

  • Your web browser will open automatically
  • You'll see the Omnara login page
  • Create a free account or sign in
  • Click "Authenticate Local CLI" button
💡 Free Tier: Omnara is free for up to 10 AI agent sessions per month. Perfect for getting started!
✅ Success: You'll see "Authentication successful!" in your terminal

5Using Omnara with Claude Code

Option 1: Monitor Your Current Claude Code Session

If Claude Code is already running, simply type in a new terminal:

omnara

This connects your current Claude Code session to the Omnara dashboard!

Option 2: Start Claude Code with Remote Access

To enable remote control from your phone or another computer:

omnara serve

This will show you a URL like:

Your session is available at: https://omnara.com/session/abc123

Now you can:

  • Open this URL on your phone
  • See what Claude Code is doing
  • Answer questions when Claude needs help
  • Guide Claude through complex tasks

❓ Frequently Asked Questions

Q: Is Omnara safe? Can it access my code?

A: Omnara only monitors the interaction between you and Claude Code. It doesn't directly access your code files - it just helps you communicate with Claude more effectively.

Q: What's the difference between "omnara" and "omnara serve"?

A: "omnara" monitors an existing Claude session, while "omnara serve" starts a new session that can be controlled remotely.

Q: Do I need to keep my terminal open?

A: Yes, the terminal running Omnara needs to stay open for the connection to remain active.

Q: Can I use this with Cursor or other AI tools?

A: Yes! Omnara supports multiple AI coding assistants including Cursor, GitHub Copilot, and more.

Q: What if I get an error during installation?

A: Try using "pip3" instead of "pip", or add "--user" to the command: pip install --user omnara
